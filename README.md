# Crystal Collector Game

You can only take home your crystals if your collection is equal to the head miner's! Blackjack style gameplay.

jQuery and dynamically updated HTML through user click events.

====================================================================================

**Technologies Used**
- HTML
- Javascript
- jQuery
- CSS
